E-commerce API Documentation

Get All Products

Endpoint: GET /api/products

Returns a list of all products in the database.

Search Products by Name

Endpoint: GET /api/products/search?name=keyword

Search products by name. Replace 'keyword' with your search term.

Get Products by Category

Endpoint: GET /api/products/category/category_name

Get all products in a specific category. Replace 'category_name' with the desired category.

Chat with Bot

Endpoint: POST /api/chat

Send a message to the chatbot. Include a JSON body with a 'message' field.
